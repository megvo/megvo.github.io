How to Embed Code

  1. Drop your scripts in assets/js/ and link them in index.html.
  2. For D3, import from a CDN and render into any div (e.g., #d3-chart).
  3. For p5, include the p5 CDN and initialize a sketch in p5-sketch.js.
  4. Use iframes for platforms that provide shareable links (Observable, Figma, Framer).
<!-- D3 from CDN -->
<script src="https://cdn.jsdelivr.net/npm/d3@7"></script>
<script src="assets/js/d3-demo.js"></script>

<!-- p5 from CDN -->
<script src="https://cdn.jsdelivr.net/npm/p5@1/lib/p5.min.js"></script>
<script src="assets/js/p5-sketch.js"></script>
